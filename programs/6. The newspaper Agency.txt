The newspaper Agency

Each Sunday, a newspaper agency sells w copies of a special edition newspaper 
for Rs.x per copy. The cost to the agency of each newspaper is Rs.y. 
The agency pays a fixed cost for storage, delivery and so on of Rs.100 per Sunday. 
The newspaper agency wants to calculate the profit which it obtains only on Sundays. 
Can you please help them out by writing a program to compute the profit 
if w, x, and y are given.
INPUT FORMAT:

Input consists of 3 integers:

w, x, and y.

w is the number of copies sold, x is the cost per copy and y is the cost the agency spends per copy.

OUTPUT FORMAT:

The output consists of a single integer which corresponds to the profit obtained by the newspaper agency.

SAMPLE INPUT: 

1000

2

1

SAMPLE OUTPUT: 

900


inputs:

case 1         case 2         case 3
1000           1200           200
2              23             3
1              4              1



program:


#include<iostream>
using namespace std;
int main()
{
  int w,x,y,profit;
  std::cin>>w>>x>>y;
  profit= (w*x)-(y*w)-100;
  std::cout<<profit;
  return 0;
  
  
}
