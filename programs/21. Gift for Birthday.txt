Gift for Birthday


Lucy is celebrating her 15th birthday. Her father promised her that he will buy her a new 
computer on her birthday if she solves the question asked by him. He asks Lucy to 
find out whether the year on which she was born is a leap year or not. 
Help her to solve this puzzle so that she celebrates her birthday happily. 
If her birth year is 2016 and it is a leap year display “2016 is a leap year.” 
Else display “2016 is not a leap year.”

INPUT & OUTPUT FORMAT:

Input consist of 1 integer.

Output consist of 1 string.

SAMPLE INPUT:

2016

SAMPLE OUTPUT:

2016 is a leap year


inputs:

case 1         case 2         case 3
2016           1999           1546



program:

#include<iostream>
using namespace std;
int main()
{
  int birth_year;
  cin>>birth_year;
  if(birth_year%4==0)
  {
    if(birth_year%100==0)
    {
      if(birth_year%400==0)
      {
       cout<<birth_year<<" is a leap year" ;
      }
      else
      {
        cout<<birth_year<<" is not a leap year";
      }
    }
    else
    {
      cout<<birth_year<<" is a leap year";
    }
  }
  else
  {
    cout<<birth_year<<" is not a leap year";
  }
  return 0;
}
